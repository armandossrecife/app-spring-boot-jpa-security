Sobre a aplicação:

Bem Vindo a aplicação Exemplo Básico Spring MVC

Nessa aplicação vamos trabalhar os princípios de uma aplicação de controle de acesso com o framework Spring MVC usando o padrão Spring Boot

As seguintes features estão disponíveis:

  1. Inserir Convidados
  2. Listar Convidados
  3. Alterar Convidado
  4. Remover Convidado
  5. Inserir Usuário
  6. Listar Usuários
  7. *Buscar Usuários
  8. Alterar Usuário
  9. Remover Usuário
  10. *Validação de dados no lado servidor
  11. Validação de dados no lado cliente
  12. Manipulação de dados via ORM/JPA
  13. *Controle de sessão do usuário via session
  14. *Inserção de arquivo via Multipart armazenado no filesystem do Servidor
  15. *Exibição de arquivo via Multipart armazenado no filesystem do Servidor
